Detailed Description

template<typename T, typename Compare = std::less<T>, typename Allocator = std::allocator<T>>
requires std::predicate<Compare, const T&, const T&>
class jump_list< T, Compare, Allocator >

A templated associative container implementing a Skip List data structure.

This class provides a container that stores elements in a sorted order, allowing for efficient insertion, deletion, and lookup operations. It offers similar functionality to std::set or std::map (when T is a std::pair), but is based on a Skip List for its underlying structure.

Template Parameters
TThe type of elements stored in the jump_list.
CompareA comparison function object type, which defaults to std::less<T>. It defines the ordering criterion for elements.
AllocatorAn allocator type, which defaults to std::allocator<T>. It defines the memory allocation strategy.
